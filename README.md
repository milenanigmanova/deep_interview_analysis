# Оценка цен покупателем
Данное исследование выполнено в рамках домашнего задания по дисциплине "Стратегии ценообразования".

**Цель данного исследования** – определить, каким образом покупатель производит оценку цены товаров различных категорий с помощью метода глубинного интервью. 

## Задачи исследования:
•	Выявить основные факторы, влияющие на восприятие цены покупателем; \
•	Определить, как покупатель формирует субъективное представление о «справедливой» цене;\
•	Проанализировать критерии, которыми руководствуется покупатель при оценке цен;\
•	Исследовать, какую референтную базу оценки применяет покупатель;\
•	Установить, на какие референтные цены — внутренние или внешние — опирается покупатель;\
•	Сравнить особенности поведения покупателя при оценке цен:\
  o	на товары повседневного спроса,\
  o	на товары, приобретаемые редко, но хорошо знакомые,\
  o	на новые (ранее не покупавшиеся) товары;\
•	Сформулировать гипотезы относительно восприятия цен покупателем.\
